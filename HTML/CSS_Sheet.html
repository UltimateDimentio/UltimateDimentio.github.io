CSS Cheat Sheet

CSS stands for Cascading Style Sheets, it can control multiple web pages' layouts, and it will describe how HTML elements are displayed on screen, paper, or any other media.

CSS is among the core languages of the open web and is standardized across web browsers according to W3C specifications.

Formatting

Elements Description
HTML Element HTML elements can be selected by just typing their name without brackets.
#Name An HTML Element with an ID can be selected by using # and then its name next to it.
.classname An HTML Element with a class name can be selected by typing a . next to its name.
To actual write CSS code, curly brackets {} must be typed after the element name that needs to be modified.

Basic Properties

Properties Description
background-color: It will give a background color to the desired element. A valid color name, hex code, or RGB code can be specified.
border: Here it can be specified the thickness of the border, the border style, and its color. Example: 5px solid red
width: Width is the core of the box model, which is the size of where the content of an element will be allocated.
padding: Padding is the space between the element content and the border.
border: As its name suggest, it is the border of a HTML element.
marging: Marging is the space between the border content and any other element around an specified box.
color: It is the color of an element's text.
text-align: To align text.
font-family: To give a font to an element.
border-radius: It gives roundness to the border.
text-decoration: It can be used to add or remove an underline in a link element. Useful to customize link reactions when hovering for example.

Flex Box

For more information about CSS, visit w3schools.com
Properties Description
display: Display is for controlling the layout and it will determine if an element will be black or an inline element.
flex-direction: Sets the display-direction of flex items.
flex-wrap Specifies whether the flex items should wrap or not. Useful for mobile views.
justify-content: Align flex items when they do not use all the horizontal space available.
align-items: Align flex items when they do not use all the vertical space available.
align-content: Align flex lines when there is extra space in both vertical space and items wrap.
