HTML Cheat Sheet

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.


"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

For more information, visit Developer Mozilla Org
Elements Description
<!doctype html> It is a required element that prevents a browser to switch to "quirks mode" when rendering a document.
<head> Contains machine-readable information, or metadata, about the document.
<link> States the relationship between the website and any external resource,
useful to link elements to CSS like font styles.
<title> Defines the website's title shown in page's tab.
<body> Represent the HTML content of the document.
<h1>,<h2>,<h3>,
<h4>,<h5>,<h6>,		 The six levels heading sections. <h1> is the highest, and <h6> the lowest.
<div> The div element is useful for CSS styling and Javascript functions. It won't change the HTML content until using CSS or Javascript.
<ol> Represents an ordered list. The default setting is numbered list.
<ul> Represents an unordered list. The defaul setting is dot list.
<li> Represents an element in a list from a <ol> or <ul> parent.
<a> With href attribute, the element creates a hyperlink to web pages, files, email addresses, locations within the current page, and more.
<br> Produces a line break in text.
<i> To italicize text.
<strong> The content have a strong importance. It is usually rendered in bold type.
<img> embeds an image into the website.
<table> Represents a tabular data (a two dimensional table with rows and columns).
<thead> The first row of the table that represents the head of the table.
<tbody> It represents the body of the table.
<tr> Defines a row of cells in a table.
<th> It is the header of a cells' group.
<td> It defines a table's cell that contains data.
<fieldset> Used to group several controls within a web form.
<legend> Represents for the content of its parent <fieldset>
<label> Represents a caption for the content of the <fieldset> parent
<input> To input various types of data. There are a ton of types attributes like button, date, or email.
<select> Represents a control that provides a menu of options.
<option> To define an item in a <select>, <optgroup>, and <datalist> element
<button> An interactive element activated by a user that once activated, it will perform an action.
