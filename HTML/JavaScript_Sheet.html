JavaScript Cheat Sheet

JavaScript is a scripting language that is used in online browsers to modify and update both HTML and CSS elements within a website.

For more information about JavaScript, visit w3schools.com
Elements Description
document.
getElementById()		 The method can access to an HTML element by its id.
console.log(content) the method prints a message to the console and it is mostly used to test functions and methods.
.innerHTML It will rewrite into an HTML element. It does not metter if there is already text inside the element.
let name, const name Declares a block variable, or a block const. These will only work inside their function scope and they cannot be redeclared.
if (condition), else A condition statement with code to be executed if the condition is true.
for (exp 1; exp 2; exp 3) A loop which contains a variable, condition, and an expression that is executed after each loop.
for (let variable of iterable) A loop that will run through the values of an iterable object. For example: each character of an string, or each element of an array.
while (condition) A loop that is executed as long as the condition is true. It runs from 0 to x number of times until condition is false.
do {} while(condition) A loop that is executed as long as the condition is true. It will run one time before checking the while condition.
==, === Equal to, equal value and type. Useful for declaring conditions.
let text = `text` Using `` when declaring a string can be helpful when declaring a multiline strings, or declare variables inside a string using ${...}
.concat() joins two or more strings.
.slice() Extracts a part of a string and returns the extracted part in a new string.
.trim() Removes whitespace from both sides of a string.
.replace("original text", "new text") Replaces a specified value with another value in a string.
const arr = [1, 2, "3"] JavaScript can store arrays using a const variable (since the values will not change). These arrays can store multiple types of data.
const obj = {
property1: "1",
property2: "2",
property3: "3",
}; 		JavaScript can use objects to store data.
The data stored can be a property value or a function method.
Object constructor Sometimes we need to create many objects of the same type, for that Object constructor can be used. For more information visit w3schools
